# vuejs-naija

## Project setup
```
npm install
```

### Compiles and hot-reloads for development
```
npm run serve
```

### Compiles and minifies for production
```
npm run build
```

### Lints and fixes files
```
npm run lint
```


## Contribution Guide
- Create a branch from any issue(s)
- Push Commits to that branch
- Create a PR to `develop`
